# Middlewar game #

`git clone git@bitbucket.org:middlewar/middlewar-game.git`

`npm install`

```
cp src/config.js.dist src/config.js
cp core/config.js.dist core/config.js
```

## Game UI ##

Run `npm run dev`

Open `http://localhost:3000`

Enjoy !

## AI ##

### Detailed logs ###
Run `node ai/remoteDev`

Open `http://localhost:8000`

### Run AI ###
Run `npm run ai`

Run `npm run run-ai`
